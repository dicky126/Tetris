Tetris - Readme

Short Playing Time
For both large size and small size Tetris, the time allowed is only 1 minute for each game. When time is up, the game will end and redirect to the start page of Tetris. This would prevent the elderly to spend too much time on play it.

Plain Game Layout
The game page is not designed in a colorful way. Instead, the game layout is quite plain. This makes the elderly feel comfortable when they play this game. Otherwise, eyes may easily feel tired.
